## 分支

- 列出所有分支  
`git branch` 

- 创建分支  
`git branch (branchname)`  
可以通过以下命令**创建**并**切换**到新分支  
`git checkout -b (branchname)`

- 切换分支  
`git checkout (branchname)` 

- 合并分支  
`git merge (the branchname you want merge to current branch)`

- 删除分支
- `git branch -d (branchname)`
